# Quantitative-Market-Analysis
Python-based financial market analysis and data visualization project

## Overview
This project analyzes historical stock market data using Python to understand price trends, volatility, and moving averages.

## Objectives
- Fetch real financial market data
- Perform time-series analysis
- Compute daily returns
- Visualize trends and moving averages

## Tools Used
- Python
- Pandas
- Matplotlib
- yfinance
- Jupyter Notebook

## Key Analysis
- Stock price trend visualization
- Daily returns calculation
- 20-day moving average analysis

## Dataset
Real-time financial data retrieved using Yahoo Finance API via `yfinance`.
